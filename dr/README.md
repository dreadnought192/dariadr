# Сайт-приглашение на день рождения

## Структура проекта

```
dr/
├── index.html          # Единая страница с прокруткой (скроллинг)
├── styles.css          # Общие стили
├── calendar.css        # Стили для календаря
├── details.css         # Стили для секции деталей
├── invitation.css      # Стили для секции приглашения
├── images/             # Папка для изображений
│   ├── hero-image.jpg
│   ├── restaurant-photo.jpg
│   ├── time-icon.png
│   ├── location-icon.png
│   └── dresscode-icon.png
└── README.md
```

## Структура страницы

Сайт представляет собой одну страницу с вертикальной прокруткой, состоящую из следующих секций (сверху вниз):

1. **Hero секция** - Фото с букетом роз и текстом "BIRTHDAY PARTY" / "Meruyert's birthday"
2. **Календарь** - Календарь на сентябрь 2023 с выделенными воскресеньями и временем 20:00
3. **Детали** - Место проведения, дресс-код и ссылки на социальные сети
4. **Приглашение** - Текст приглашения на русском языке
5. **Фото** - Фотография в ресторане

## Необходимые изображения

Поместите следующие изображения в папку `images/`:

### 1. `hero-image.jpg`
- **Описание**: Фотография женщины с букетом красных роз на лестнице
- **Размеры**: Рекомендуется 1200x1600px или больше (вертикальная ориентация)
- **Формат**: JPG, PNG
- **Содержание**: 
  - Молодая женщина в черном платье с открытыми плечами
  - Большой букет красных роз в белой обертке
  - Лестница с темными перилами
  - Светлые шторы на фоне
  - Текст "BIRTHDAY PARTY" и "Meruyert's birthday" будет наложен поверх изображения

### 2. `restaurant-photo.jpg`
- **Описание**: Фотография женщины в ресторане/кафе
- **Размеры**: Рекомендуется 1200x800px или больше (горизонтальная ориентация)
- **Формат**: JPG, PNG
- **Содержание**:
  - Молодая женщина сидит в ресторане/кафе
  - Длинные волнистые темно-каштановые волосы
  - Черный топ с открытыми плечами
  - Светлые джинсы
  - В руке стакан с зеленым напитком и черной соломинкой
  - Интерьер ресторана с декоративными элементами

### 3. `time-icon.png`
- **Описание**: Иконка времени (часы)
- **Размеры**: Рекомендуется 60x60px или больше (квадратное изображение)
- **Формат**: PNG (с прозрачным фоном), JPG
- **Содержание**: Иконка часов/времени

### 4. `location-icon.png`
- **Описание**: Иконка места/локации
- **Размеры**: Рекомендуется 60x60px или больше (квадратное изображение)
- **Формат**: PNG (с прозрачным фоном), JPG
- **Содержание**: Иконка места/геолокации

### 5. `dresscode-icon.png`
- **Описание**: Иконка дресс-кода (одежда)
- **Размеры**: Рекомендуется 50x50px или больше (квадратное изображение)
- **Формат**: PNG (с прозрачным фоном), JPG
- **Содержание**: Иконка одежды/дресс-кода

## Используемые шрифты

Сайт использует следующие шрифты из Google Fonts:

1. **Dancing Script** - для названия месяца "September" (скриптовый шрифт)
2. **Playfair Display** - для текста "Meruyert's birthday" (элегантный шрифт с засечками)
3. **Inter** - основной шрифт для всего остального текста (современный sans-serif)

Все шрифты загружаются автоматически через Google Fonts.

## Навигация

Сайт представляет собой одну страницу с плавной вертикальной прокруткой. Стрелка вниз на первой секции (hero) указывает на возможность прокрутки вниз.

## Цветовая схема

- **Основной текст**: #2c2c2c (темно-серый)
- **Акценты (воскресенья)**: #d32f2f (красный)
- **Фон**: #ffffff (белый)
- **Иконки соцсетей**: #2c2c2c (темно-серый фон, белые иконки)

- Адаптивный дизайн для мобильных устройств
- Плавная анимация навигационных стрелок

- Поддержка русского и английского языков
